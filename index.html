Fallacies of JS Performance

  • Obfuscating is enough optimization.
  • Objects are hashmaps.
  • Environment variables are constant.
  • Stack frames are free.
  • JS logic should be as sync as possible.
  • The JS runtime is single-threaded.
  • JS code is multi-threaded.
