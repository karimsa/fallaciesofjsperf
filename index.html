Fallacies of JS Performance

  • Minification is for the front-end.
  • Bundling is for the front-end.
  • Inheritance is free.
  • Objects are hashmaps.
  • Environment variables are constant.
  • Stack frames are free.
  • Logic should be as async as possible.
  • The JS runtime is single-threaded.
  • SSR is only for SEO.
